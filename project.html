This is an IT project

This is an LMS project, which combines a stable technical framework and database to achieve the purpose of managing user learning. Users can make their own learning plans through the LMS platform. The platform can make curriculum for users, and there are many book resources and teacher resources for users to learn.

The system will be tested many times before it goes online. Testing is only to give users the best experience, and each part of the system also selects the best in the market.
